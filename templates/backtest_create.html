2. Signaux d’entrée & de sortie

Un signal est un déclencheur : il définit quand une configuration BUY ou SELL devient vraie. Les filtres et conditions de marché ci-dessous peuvent encore bloquer un BUY même si les signaux sont valides.

Chaque ligne peut combiner 1..N conditions BUY et 1..N conditions SELL. Par défaut : BUY = ET, SELL = OU. Le mode d’exécution contrôle comment les positions sont ouvertes et clôturées dans le temps.
Résumé de la stratégie

ℹ️
Mode de stratégie
Choisis Signaux classiques pour utiliser les lignes BUY/SELL habituelles, ou Stratégie Couloir pour utiliser le système autonome achat/vente.
Configuration Couloir
Couloir achète après un rebond depuis un plus bas suivi, puis vend après un repli depuis le plus haut atteint depuis l’achat.
En V1, Couloir est une stratégie autonome achat/vente. Elle est conçue pour être utilisée seule comme logique principale BUY/SELL. Les filtres et protections GM/GM_push restent compatibles, mais les autres signaux BUY/SELL classiques sont ignorés lorsque Couloir est actif.
