Aide indicateurs

Version nettoyée du moteur : les indicateurs actifs sont P, δj, M/X, M1/X1, T, Q, S, K1, K2, K3, K4, Kf, SUM_SLOPE, SLOPE_VRAI, le signal anti-chute RHD, et les signaux associés.

Prix de référence

P(t) = (a·F(t) + b·H(t) + c·L(t) + d·O(t)) / (a+b+c+d)

Rendement journalier

δj(t) = (P(t) - P(t-1)) / P(t-1)

Legacy conservé

M(t)=max(P(t-1)...P(t-N1))
X(t)=min(P(t-1)...P(t-N1))
M1(t)=moyenne sur N2 de M
X1(t)=moyenne sur N2 de X
T(t)=(M1(t)-X1(t))/e
Q(t)=M1(t)-T(t)
S(t)=M1(t)+T(t)
K1=P-M1
K2=P-X1
K3=P-Q
K4=P-S

Kf

p(t) = somme sur N2 de δj
Kf(t) = M1(t) - T(t)·p(t)

SUM_SLOPE

SUM_SLOPE(t) = somme sur Npente de δj(t)

SLOPE_VRAI

SLOPE_VRAI(t) = (P(t) - P(t-Npente)) / P(t-Npente)

Signaux actifs

  • A1 / B1
  • C1 / D1
  • E1 / F1
  • G1 / H1
  • Af / Bf
  • SPa / SPv
  • SPVa / SPVv
  • SPa_basse / SPv_basse
  • SPVa_basse / SPVv_basse
  • RHD_OK / RHD_FAIL

Pour les pentes, les signaux BUY (SPa / SPVa, et variantes basses) croisent le seuil d’achat vers le haut. Les signaux SELL (SPv / SPVv, et variantes basses) croisent le seuil de vente vers le bas. Ce n’est pas un miroir autour d’un seuil négatif. Si le seuil de vente est vide, le seuil d’achat est réutilisé et le comportement historique est conservé.

Signal anti-chute RHD — Repli depuis haut récent

Pmax_X(t) = max(P(t-1) ... P(t-X))
seuil(t) = Pmax_X(t) * (1 + Y)
condition vraie si P(t) >= seuil(t)

Le jour courant est exclu du calcul du plus haut récent. Exemple : avec X=10 et Y=-10 %, si le plus haut des 10 jours précédents vaut 100, RHD_OK est actif tant que le prix du jour reste au-dessus de 90 ; RHD_FAIL est émis lorsque cette limite est dépassée.

RHD_OK et RHD_FAIL sont de vrais signaux ticker calculés. Ils peuvent être utilisés dans les règles BUY et SELL comme les autres signaux ticker.

Ce n’est pas un stop-loss basé sur le prix d’achat, et ce n’est pas un filtre GM de marché.

Supprimé de l'interface et du moteur actif : K1f, K2f, Kf3, ligne V, signaux A1f/B1f, A2f/B2f, AF3/BF3, I1/J1, ainsi que les variables N3, N4, N5, VC, FL, cr, N5f3, crf3, NampL3, baseL3, periodeL3 et m_v.
