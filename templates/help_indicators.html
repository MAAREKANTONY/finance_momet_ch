Aide indicateurs

Version nettoyée du moteur : les indicateurs actifs sont P, δj, M/X, M1/X1, T, Q, S, K1, K2, K3, K4, Kf, SUM_SLOPE et les signaux associés.

Prix de référence

P(t) = (a·F(t) + b·H(t) + c·L(t) + d·O(t)) / (a+b+c+d)

Rendement journalier

δj(t) = (P(t) - P(t-1)) / P(t-1)

Legacy conservé

M(t)=max(P(t-1)...P(t-N1))
X(t)=min(P(t-1)...P(t-N1))
M1(t)=moyenne sur N2 de M
X1(t)=moyenne sur N2 de X
T(t)=(M1(t)-X1(t))/e
Q(t)=M1(t)-T(t)
S(t)=M1(t)+T(t)
K1=P-M1
K2=P-X1
K3=P-Q
K4=P-S

Kf

p(t) = somme sur N2 de δj
Kf(t) = M1(t) - T(t)·p(t)

SUM_SLOPE

SUM_SLOPE(t) = somme sur Npente de δj(t)

Signaux actifs

  • A1 / B1
  • C1 / D1
  • E1 / F1
  • G1 / H1
  • Af / Bf
  • SPa / SPv

Supprimé de l'interface et du moteur actif : K1f, K2f, Kf3, ligne V, signaux A1f/B1f, A2f/B2f, AF3/BF3, I1/J1, ainsi que les variables N3, N4, N5, VC, FL, cr, N5f3, crf3, NampL3, baseL3, periodeL3 et m_v.
